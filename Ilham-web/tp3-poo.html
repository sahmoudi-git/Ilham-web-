            Exercice 1: Polymorphisme et Redéfinition des Méthodes

            1. Définissez une classe abstraite Vehicule avec une méthode
            
            2. Créez des classes dérivées Voiture et Avion qui héritent de Vehicule.
            
            3. Redéfinissez la méthode deplacer() dans chaque classe dérivée pour afficher un message spécifique à chaque type de véhicule.
            
            4. Créez une méthode statique prime() qui prends en paramêtre un nombre entier positif et qui détermine s'il s'agit d'un 

                nombre premier ou non.
            from abc import ABC, abstractmethod

            class Vehicule(ABC):
                @abstractmethod
                def deplacer(self):
                    pass
            
            class Voiture(Vehicule):
                def deplacer(self):
                    print("La voiture se déplace sur la route.")
            
            class Avion(Vehicule):
                def deplacer(self):
                    print("L'avion vole dans le ciel.")
            
                @staticmethod
                def prime(n):
                    if n < 2:
                        return False
                    for i in range(2, int(n**0.5) + 1):
                        if n % i == 0:
                            return False
                    return True
            
            voiture = Voiture()
            voiture.deplacer()
            
            avion = Avion()
            avion.deplacer()
            
            print(Avion.prime(7))  
            print(Avion.prime(10)) 
                
        
            Exercice 2: Surcharge des Opérateurs et Utilisation des Méthodes Abstraites

            1. Définissez une classe abstraite Forme avec une méthode abstraite calculer_surface().
            
            2. Créez des classes dérivées Rectangle et Cercle qui héritent de Forme.
            
            3. Redéfinissez la méthode calculer_surface() dans chaque classe dérivée pour calculer la surface spécifique à chaque forme.
            
            4. Créez une méthode statique somme_aires() dans chaque classe dérivée qui renvoie la somme des surfaces de 2 formes.
            
                from abc import ABC, abstractmethod
                import math
                
                class Forme(ABC):
                    @abstractmethod
                    def calculer_surface(self):
                        pass
                
                class Rectangle(Forme):
                    def __init__(self, largeur, longueur):
                        self.largeur = largeur
                        self.longueur = longueur
                
                    def calculer_surface(self):
                        return self.largeur * self.longueur
                
                    @staticmethod
                    def somme_aires(rect1, rect2):
                        return rect1.calculer_surface() + rect2.calculer_surface()
                
                class Cercle(Forme):
                    def __init__(self, rayon):
                        self.rayon = rayon
                
                    def calculer_surface(self):
                        return math.pi * (self.rayon ** 2)
                
                    @staticmethod
                    def somme_aires(cerc1, cerc2):
                        return cerc1.calculer_surface() + cerc2.calculer_surface()
                
                rect1 = Rectangle(2, 3)
                rect2 = Rectangle(4, 5)
                print(rect1.calculer_surface())  
                print(rect2.calculer_surface())  
                print(Rectangle.somme_aires(rect1, rect2))  
                
                cerc1 = Cercle(3)
                cerc2 = Cercle(4)
                print(cerc1.calculer_surface())  
                print(cerc2.calculer_surface())  
                print(Cercle.somme_aires(cerc1, cerc2))
